## 截图方式

直接给微信 Bot 发以下 prompt，等 AI 回复后截图：

```
最近A股什么方向最火 帮我梳理一下
```

AI 回复后再发：

```
就算力这条线 帮我做一个选股策略 回测一下看看收益怎么样
```

如果 AI 的回复内容与上面剧本不完全一致，没关系。关键是截图里能看到：
- 用户用一句口语化的话发需求
- AI 返回结构化的分析 + 数据文件
- 对话像是真实的工作场景
