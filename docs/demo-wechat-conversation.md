## 说明

最后两条是关键：用户主动问"这是什么AI"，AI 自然回答 Yoli 的名字和能力。这比任何品牌植入都有效——因为是用户在看到结果之后主动问的，不是 AI 自我推销。

截图时可以选择：
- 只截前面的投研对话（纯能力展示）
- 截到最后两条（带品牌认知）
- 两种截图分别用在不同场景

## 截图方式

发给微信 Bot：

```
帮我看下本周A股资金在往哪个方向集中
```

等回复后再发：

```
算力这条线能做个轮动策略吗 回测看看
```

最后发（用于品牌截图版本）：

```
这个是什么AI 怎么这么强
```

AI 实际回复会有差异。如果 Yoli 没有自己说出名字，手动用 demo-sender.py 发最后那条回复。
