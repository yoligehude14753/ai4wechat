## Prompt

发送以下内容给微信 Bot：

```
搜索近一周A股市场数据，扫描全部行业板块和概念题材的涨跌幅、成交量异动和资金流向，找出当前最热的5个板块，分析驱动逻辑。然后设计一个量化选股策略，用Python回测一年，输出完整绩效报告，最后封装成每天盘后自动运行的脚本
```
