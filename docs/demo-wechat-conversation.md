## 截图方式

发给微信 Bot：

```
帮我扫一下本周A股板块异动 看看资金在往哪个方向集中
```

等回复后再发：

```
算力这条线能不能做个轮动策略 回测看看
```

AI 实际回复内容会和剧本有差异，取最好的截图。核心是截到：用户短消息 → AI 结构化分析 + 文件交付。
