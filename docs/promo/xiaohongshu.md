ai4wechat — 让你的 AI 产品在微信里被使用

你的 AI 服务已经做好了，但用户想在微信里直接用。

ai4wechat 帮你搞定。安装一行命令，微信扫码，用户就能在微信里和你的 AI 对话。你的服务不需要改任何东西。

怎么用：
1. pip install ai4wechat
2. ai4wechat-serve --target-url 你的AI服务地址
3. 扫码
4. 完成

支持所有有 HTTP 接口的 AI 服务。ChatGPT 封装、客服 AI、知识库问答、Agent 都行。

大模型输出的 Markdown 会自动转成微信能读的格式。超长消息自动分段。服务器部署支持远程扫码。
