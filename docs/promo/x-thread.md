ai4wechat — make your AI product usable inside WeChat.

Your AI service already has an HTTP endpoint. ai4wechat bridges it to WeChat.

pip install ai4wechat
ai4wechat-serve --target-url http://your-service/chat

Scan QR. Users can now talk to your AI from WeChat.

Open source: github.com/yoligehude14753/ai4wechat

What it does:

· Bridges WeChat messages to your existing AI service via HTTP
· Auto-converts LLM Markdown output to clean WeChat text
· Splits long messages with page numbers
· Typing indicator while AI processes
· Web login for remote servers

Also works as a Python SDK with @bot.on_message decorator.

Your service needs zero changes. MIT license.
