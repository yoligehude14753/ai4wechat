# 截图制作操作手册

不停 Yoli，不影响任何服务。脚本只负责发送预设回复。

---

## 操作流程

### 1. 传脚本到服务器

```bash
gcloud compute scp ~/Desktop/ai4wechat/docs/demo-sender.py yoli-server-2:~/demo-sender.py --zone=us-central1-a --tunnel-through-iap
```

### 2. 在服务器上运行

```bash
gcloud compute ssh yoli-server-2 --zone=us-central1-a --tunnel-through-iap

pip3 install httpx  # 如果没装过
python3 ~/demo-sender.py
```

### 3. 手机发一条消息给 Bot

随便发什么都行，比如"你好"。脚本需要这条消息来获取你的 user_id。

发完后回到终端按回车。

### 4. 逐条发送预设回复

脚本会依次显示 7 条预设回复。每条：

1. 你先在手机微信里按脚本发你的消息
2. Yoli 会自动回复一条（**忽略它**）
3. 回到终端按回车 → 脚本发送预设回复
4. 微信里出现预设回复（和 Yoli 的回复并列）
5. **长按 Yoli 的自动回复 → 删除** → 只剩预设回复
6. 截图

### 5. 完事

Ctrl+C 退出脚本。Yoli 全程没有受影响。

## Yoli 自动回复怎么处理

Yoli 收到你的消息后会自动回复 AI 生成的内容。处理方式：

- **方法 1（推荐）**：长按 Yoli 的自动回复 → "删除" → 只剩预设回复，再截图
- **方法 2**：截图后用图片编辑裁掉 Yoli 的回复
- **方法 3**：无所谓，Yoli 的回复反而让截图更真实（展示 AI 的实际能力）

## 发文件的截图

如果需要截图里出现文件（比如"技术要求响应文件_v1.docx"）：

1. 用电脑微信的文件传输助手，把 `/tmp/技术要求响应文件_初稿_v1.docx` 发到手机
2. 在和 Bot 的对话里，从"最近文件"里转发这个 docx
3. 微信里会显示文件图标 + 文件名 + 大小（12MB）

这样截图里就有真实的文件卡片了。
