# 截图操作手册

照做就行，10 分钟搞定。

---

## 第 1 步：把脚本传到服务器

在你的 Mac 终端执行这一条命令：

```bash
gcloud compute scp ~/Desktop/ai4wechat/docs/demo-sender.py yoli-server-2:~/demo-sender.py --zone=us-central1-a --tunnel-through-iap
```

---

## 第 2 步：SSH 到服务器运行脚本

```bash
gcloud compute ssh yoli-server-2 --zone=us-central1-a --tunnel-through-iap
```

连上后：

```bash
pip3 install httpx
python3 ~/demo-sender.py
```

看到这个界面就对了：

```
==================================================
  ai4wechat 截图工具
==================================================

  共 3 条预设回复

  操作流程：
  1. 你在手机微信里发消息
  2. Yoli 会自动回复（长按删掉）
  3. 回到这里按回车 → 发送预设回复
  4. 手机截图
```

---

## 第 3 步：初始化

脚本提示你先发一条消息。

**手机操作**：打开微信，给 Bot 发一条"你好"。

**终端操作**：按回车。

看到 `✓ 已获取` 就行。

Yoli 会自动回复你的"你好"，**长按那条回复 → 删除**。

---

## 第 4 步：截图第一轮

**手机操作**：发这条消息：

```
帮我看下本周A股资金在往哪个方向集中
```

Yoli 会自动回复一条。**长按删掉**。

**终端操作**：按回车。

你的微信里会出现预设的板块分析回复。

---

## 第 5 步：截图第二轮

**手机操作**：发这条消息：

```
算力这条线能做个轮动策略吗 回测看看
```

Yoli 自动回复，**长按删掉**。

**终端操作**：按回车。

微信里出现策略回测结果。

---

## 第 6 步：截图第三轮

**手机操作**：发：

```
部署
```

Yoli 自动回复，**长按删掉**。

**终端操作**：按回车。

微信里出现部署确认。

---

## 第 7 步：截图

在手机上截图。完事。

脚本按 q 或 Ctrl+C 退出。Yoli 不受影响。

---

## 注意

- 每次你发消息，Yoli 都会自动回复一条 → **长按删掉**再截图
- 预设回复是纯文本，文件附件（csv/py）在截图里不会出现 — 如果需要文件卡片效果，提前用文件传输助手把文件发到手机上，在对话里转发
- 如果脚本报错，直接重新跑 `python3 ~/demo-sender.py`
